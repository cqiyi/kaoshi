2003年11月7日数据库上机考试题目：

有某个学生运动会比赛信息的数据库，保存了如下的表：
运动员（运动员编号，运动员姓名，运动员性别，所属系号）
项目（项目编号，项目名称，项目比赛地点）
成绩（运动员编号，项目编号，积分）
请用SQL语句完成如下功能：

1. 建表，注意满足如下要求：
a) 表名必须加自己的帐号作前缀，比如“d10000001运动员”，
b) 定义各个表的主码外码约束
c) 运动员的姓名和所属系别不能为空值。
d) 积分要么为空值，要么为6,4,2,0, 分别代表第一，二，三名和其他名次的积分，注意名次可以有并列名次，后面的排名不往前提升，例如， 如果有两个并列第一，则没有第二名。

2. 往表中插入数据：
运动员（ 1001, 李明，男，计算机系
   1002，张三，男，数学系
   1003, 李四，男，计算机系
   1004, 王二，男，物理系
   1005, 李娜，女，心理系
   1006, 孙丽，女，数学系）
项目（ x001, 男子五千米，一操场
   x002，男子标枪，一操场
   x003, 男子跳远，二操场
   x004, 女子跳高，二操场
   x005, 女子三千米，三操场）
积分（  1001,  x001, 6
        1002,   x001, 4
        1003,  x001, 2
        1004,  x001, 0
        1001,  x003, 4
        1002,  x003, 6
        1004,  x003, 2
        1005,  x004, 6
        1006,  x004, 4)

3. 完成如下查询：a) 求出目前总积分最高的系名，及其积分。
b) 找出在一操场进行比赛的各项目名称及其冠军的姓名
c) 找出参加了张三所参加的所有项目的其他同学的姓名

4. 经查张三因为使用了违禁药品，其成绩都记0分，请在数据库中作出相应修改。

5. 经组委会协商，需要删除女子跳高比赛项目。

 并用SQL语言完成以下功能：
 1 建表，在定义中要求声明：
     各表的主外键；
     运动员姓名，系别不为空；
     积分为空或6,4,2,0分别代表第一名，第二名，第三名，
     其他名次的积分。

use pku3

create table a
(a# char(4) not null,
 aname varchar(10),
 sex char(2),
 dname varchar(8) not null,
 primary key(a#))

create table p
(p# char(4),
 pname varchar(10),
 addr varchar(8)
 primary key(p#))

create table g
(a# char(4) not null,
 p# char(4),
 grade tinyint,
 primary key(a#, p#),
 foreign key (a#) references a(a#) on delete cascade,
 foreign key (p#) references p(p#) on delete cascade,
 check(grade = NULL or grade = 6 or grade = 4 or 
       grade = 2 or grade = 0))

/*******************************************
 3Query.sql 完成以下查询功能：
 
 1)列出目前获总积分最高的系及其积分。
 2)列出在一操场进行比赛的各项目及其冠军。
 3)参加了张三所选的所有项目的同学名字。

 *******************************************/
use pku3

-- 1)列出目前获总积分最高的系及其积分。

select dname, sum(grade)as TotalGrade
from a, g
where a.a# = g.a#
group by dname
having sum(grade)>=all(select sum(grade)
                       from a, g
                       where a.a# = g.a#
                       group by dname)


-- 2)列出在一操场进行比赛的各项目及其冠军。

select aname, pname
from a, p, g
where p.p# = g.p# and a.a# = g.a#
       and addr = '一操场' and grade = 6

-- 3)参加了张三所选的所有项目的同学名字。

drop table aaa

select p# 
into aaa
from g, a
where a.a# = g.a# and aname = '张三'

-- insert into g values('1003', 'x002', '4') 
-- 此句做测试用， 加入“李四参加X002运动”
-- 这样李四是张三之外另一个参加了张三所选
-- 的所有项目的同学
delete from g where a#='1003' and p#='x002'

select distinct aname
from a
where not exists
      (select p#
       from aaa
       where not exists
             (select *
              from g
              where g.a# = a.a# and g.p# = aaa.p#))
